# Chat App Frontend

React + TypeScript + Vite + Tailwind CSS

## Prerequisites

- Bun (https://bun.sh)

## Setup

1. Install dependencies:
```bash
bun install
```

2. Configure backend URL in `.env`:
```
VITE_API_URL=http://localhost:8080/api
```

3. Start development server:
```bash
bun run dev
```

## Build

```bash
bun run build
```
